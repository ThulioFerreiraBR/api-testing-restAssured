# Desafio Técnico QA - Automação de API

## Objetivo
Este projeto tem como objetivo validar os endpoints da API DummyJSON através de testes automatizados de API utilizando Java, Rest Assured e JUnit 5.

## Tecnologias Utilizadas
- Java 17
- Maven
- JUnit 5
- Rest Assured
- GitHub Actions

## Como Executar

## Executar testes

mvn clean test
